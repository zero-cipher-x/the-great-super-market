# 🛒 The Great Super Market

Welcome to **The Great Super Market** — a modern, user-friendly e-commerce web application designed to streamline the grocery shopping experience online. Users can browse a wide variety of fresh produce, pantry staples, and household items, manage a virtual cart, and complete transactions securely.

---

## 🚀 Features

- **Interactive Product Catalog:** Filter and search items by category (e.g., Fruits, Vegetables, Dairy, Bakery).
- **Dynamic Shopping Cart:** Add, remove, and update item quantities in real-time with an instant price calculator.
- **User Authentication:** Secure sign-up, login, and logout capabilities for a personalized experience.
- **Order Management:** Track past orders, view digital receipts, and manage profile settings.
- **Responsive Design:** Optimized for seamless performance across desktops, tablets, and mobile devices.
