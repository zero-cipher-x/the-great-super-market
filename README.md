A modern full-stack supermarket management platform designed to streamline inventory management, sales tracking, customer management, and business operations.

## 📖 Overview

The Great Super Market is a web application created to help supermarkets and retail businesses efficiently manage their daily operations.

The system provides tools for managing products, inventory, sales, customers, and administrative tasks through a user-friendly interface.

This project serves as both a practical business solution and a demonstration of full-stack software development skills.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/zero-cipher-x/the-great-super-market.git
```

### 2. Navigate to the Project

```bash
cd the-great-super-market
```

### 3. Install Backend Dependencies

```bash
cd server
npm install
```

### 4. Install Frontend Dependencies

```bash
cd ../client
npm install
```

### 5. Configure Environment Variables

Create a `.env` file in the server directory:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=supermarket_db
DB_PORT=3306
PORT=5000
```

### 6. Start Backend Server

```bash
cd server
npm start
```

### 7. Start Frontend Application

```bash
cd client
npm run dev
```
